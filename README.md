# NlwIA

Um projeto front-end leve construído com **Next.js**, **React** e **Tailwind CSS**.

## ✨ O que você encontra aqui

- Layout global com **header fixo** e **footer fixo**, presente em todas as páginas.
- Editor de código integrado para testar snippets rapidamente.
- Componentes reutilizáveis organizados em `src/components/ui`.

## 🚀 Como rodar localmente

```bash
npm install
npm run dev
```

Abra o navegador em `http://localhost:3000`.

## ✅ Funcionalidades principais

- Navegação simples com links de exemplo.
- Header e Footer sempre visíveis enquanto você rola a página.
- Editor de código com visual escuro e suporte a edição.

## 📍 Onde começar a mexer

- Layout global: `src/app/layout.tsx`
- Página principal: `src/app/page.tsx`
- Componentes UI: `src/components/ui/`
